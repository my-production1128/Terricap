# TerriCap（テリキャップ）

**ゲーミフィケーションを活用した、位置情報×ヘルスケア連動型アプリケーション**

## 📝 プロジェクト概要
本プロジェクトは、スマートフォンの位置情報と歩数・カロリーデータを活用し、現実世界のスポット（公園など）を舞台に陣取りを行うiOS向けゲームアプリケーションです。
現在、チームによる**共同開発プロジェクトとして進行中（開発途中）**であり、私は**iOSアプリ側のフロントエンド実装全般**を担当しています。

複数のAppleネイティブAPI（MapKit, HealthKit, RealityKit等）をSwiftUI上で統合し、堅牢なアーキテクチャ設計とインタラクティブなUI/UXの構築に注力しています。

## 👤 担当領域と技術的工夫（フロントエンド）
UIコンポーネントの作成から、デバイスのハードウェア機能（位置情報・モーションセンサー）との連携、バックエンド（Supabase）との非同期通信まで、クライアント側の実装をリードしています。

### 1. 拡張性と保守性を意識したアーキテクチャ設計 (MVVM + Repository)
* **関心の分離:** UI層（View）と状態管理・ビジネスロジック層（ViewModel）を明確に分離。また、API通信やデータ永続化の処理はRepository層に隠蔽することで、テスト容易性と保守性の高いコードベースを構築しています。
* **状態管理の最適化:** `@Published` や `Environment` を適切に使い分け、複雑な画面間でのデータ同期（歩数ログ、現在地、ユーザー認証状態など）を安全に管理しています。

### 2. Appleネイティブフレームワークの高度な統合
* **位置情報とバックグラウンド処理 (CoreLocation / MapKit):** ユーザーの現在地をリアルタイムに監視。目標スポットから指定距離（300m）以内に入り、かつ歩数条件を満たした「瞬間」に自動で占有APIをトリガーするロジックを実装。通信の無限ループを防ぐため、条件判定と通信処理を分離し、適切なフラグ管理を行っています。
* **ヘルスケア連携 (HealthKit / CoreMotion):**
  チート防止を考慮し、CoreMotionによるリアルタイムな歩数計測と、HealthKitを通じたApple Watchからの消費カロリー集計を組み合わせて実装しています。
* **3Dモデルの動的描画 (RealityKit):**
  ユーザーの歩行実績に応じて成長する3Dキャラクターを配置。Game Centerから取得したユーザーデータ（初期値・回復率）を元に、マテリアル（肌の色や模様）を動的に変更する処理を実装しています。
* **Live Activities (ActivityKit):**
  アプリがバックグラウンドにある状態でも、ロック画面やDynamic Island上で「現在の歩数」「目標までの距離」「歩行ステータス」をリアルタイムに確認できる仕組みを導入し、UXを向上させました。

### 3. UI/UXの追求
* 画面下部を占有するハーフモーダルから、マップの視認性を損なわない「フローティングカードUI」へ設計を改善。`.transition` や `withAnimation` を駆使し、タスク達成時に直感的で心地よいフィードバックが得られるアニメーションを実装しています。

## 🛠 技術スタック (フロントエンド領域)
* **開発言語:** Swift
* **UIフレームワーク:** SwiftUI
* **アーキテクチャ:** MVVM, Repository Pattern
* **Apple Frameworks:**
  * MapKit / CoreLocation (マップ描画・位置情報トラッキング)
  * HealthKit / CoreMotion (ヘルスケアデータ取得・歩数計測)
  * RealityKit (3Dモデルレンダリング)
  * ActivityKit (Live Activities / Dynamic Island)
  * GameKit (Game Center認証・アバター取得・ランキング)
  * AuthenticationServices (Sign in with Apple)
* **BaaS / Backend:** Supabase (Swift SDKによる非同期通信・Realtime API連携)

## ✨ 主要機能（開発中）
* **リアルタイム陣取りマップ:** 現在地周辺のスポットを自動検索してピンを描画。実際にその場所へ移動し、歩数条件を満たすことで陣地を占有。
* **リアルタイム同期:** Supabaseの `RealtimeChannel` を用い、他プレイヤーが陣地を占有した際も瞬時にクライアントのマップUI（ピンの色など）を更新。
* **マルチプレイ＆ランキング:** Game Center連携によるシームレスなアカウント作成と、ユーザー間の月間・週間スコアランキング。
* **3Dアバター育成機能:** ユーザーの歩数・消費カロリーの実績値に応じて、RealityKit上のキャラクターがスケールアップする育成要素。
